Empleos Ahora

Empleos ahora is a project that seeks to bring remote work within the Mexican territory. My contribution among other things was the recruitment page. You can see the proyect here

How it was built

The application is a web system which interacts through JavaScript with the database through APIs, It also has account validation by sending email

The API is coded in an Apache and PHP server with the REST methodology, the database is a MySql server

Database

The structure of the database allows access to large segments of information in a short time since it is optimized for an immediate response

HTML & CSS

the design was made with the support of the Bootstrap documentation, modals, dropdown menus, colors and layouts are used

Mailing

to validate the identity of the prospect, the system requests an email and validates that it exists by sending a confirmation email .
